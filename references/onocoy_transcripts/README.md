- `transcript_manifest_2026-02-14.tsv`: index of transcript IDs, source metadata, and evidence tier.

## Evidence handling guidance
- Treat all transcript content as **secondary evidence** until validated by primary artifacts.
- Use transcripts for:
  - claim discovery
  - parameter hints
  - scenario/archetype framing
- Do not use transcripts alone for:
  - token supply/vesting values
  - on-chain balances/flows
  - KPI threshold calibration
  - definitive policy/financial conclusions

## Primary-source follow-up needed
For hard claims, collect:
- official docs/tokenomics PDF with release dates
